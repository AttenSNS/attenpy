## 作業フロー（必ず守ること）

### 1. タスクの整理

- ユーザーの要求を理解し、行うべきことをリストアップする
- 必要に応じてファイル構造やコードを調査する

### 2. 実装

- 下記のコーディングルールに従って実装する
- 一つの機能が完成したら次のステップへ進む

### 3. 検証

以下のコマンドを`attenpy/`上で実行して失敗しないか確認する

```
ruff check
pyright
ruff format
```
